# 用电异常告警系统 · 概要设计

## 技术架构

```
┌──────────────────────────────────────┐
│         前端 Dashboard (React + TS)   │
├──────────────────────────────────────┤
│     REST API + SSE 实时推送           │
├──────────────────────────────────────┤
│  数据采集  │  阈值评估  │  通知服务    │
├──────────────────────────────────────┤
│     SQLite 业务库 + 内存缓存          │
└──────────────────────────────────────┘
```

## 核心模块

1. **数据采集模块**: 从模拟电表采集电压/电流/功率数据
2. **阈值评估引擎**: 多级阈值(预警/警告/严重)实时评估
3. **告警通知**: 站内信 + 邮件通知，支持告警升级
4. **统计报表**: 按日/周/月统计异常事件，支持导出

## 告警流程

```
数据采集 → 阈值比对 → 触发条件满足？
                        ├─ 是 → 生成告警 → 分级通知 → 记录日志
                        └─ 否 → 丢弃(仅写时序库)
```
