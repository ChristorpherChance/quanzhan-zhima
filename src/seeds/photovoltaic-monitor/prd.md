# 光伏电站监控系统 · PRD

## 需求概述

构建一个分布式光伏电站的实时监控与运维管理平台。

## 目标用户

- 电站运维工程师（主要）
- 电站站长/管理人员
- 设备厂商技术支持

## 核心场景

1. **实时监控大屏**: 运维人员在大屏上查看所有电站的实时运行状态、关键指标（发电功率、电压、电流、逆变器状态）
2. **告警处理**: 系统检测到设备异常（如逆变器停机、温度过高、功率骤降）时自动推送告警，运维人员确认并处理
3. **巡检管理**: 运维人员按计划执行设备巡检，提交巡检记录，系统跟踪巡检完成率
4. **数据分析**: 管理人员查看日报/周报/月报，分析发电效率、设备利用率、故障率等指标

## 功能需求

### P0 (必须有)
- 设备状态实时监控（逆变器、汇流箱、电表）
- 关键指标仪表盘（功率、发电量、可用率）
- 异常告警推送与确认
- 设备列表与详情查看

### P1 (应该有)
- 巡检计划管理与记录
- 日报/月报自动生成
- 历史数据查询（曲线图）
- 告警规则配置

### P2 (可以有)
- 移动端适配
- 多电站切换
- 数据导出(Excel)
- 天气预报集成

## 非功能需求

- 数据刷新延迟 < 5 秒
- 支持 100+ 设备同时监控
- 单页面操作响应 < 500ms
