# 光伏电站监控系统 · 概要设计

## 技术架构

```
┌─────────────────────────────────────┐
│         前端 SPA (React + TS)        │
├─────────────────────────────────────┤
│          API Gateway (REST)          │
├─────────────────────────────────────┤
│  设备管理  │  告警引擎  │  统计服务   │
├─────────────────────────────────────┤
│         InfluxDB 时序数据库          │
│         SQLite 业务数据库            │
└─────────────────────────────────────┘
```

## 核心模块

1. **设备管理模块**: 设备注册、状态跟踪、参数配置
2. **数据采集模块**: 模拟设备数据源，定时采集并存储到时序库
3. **告警引擎**: 规则评估 → 告警生成 → 通知分发
4. **统计报表**: 日报/周报/月报生成，数据聚合计算
5. **前端展示**: 仪表盘、设备列表、告警列表、报表页

## 数据流

```
设备传感器 → 数据采集 → InfluxDB → 告警引擎(实时规则评估)
                                        ↓
                                   告警生成 → 通知分发
                                        ↓
                                    前端实时推送 (SSE/WebSocket)
```
