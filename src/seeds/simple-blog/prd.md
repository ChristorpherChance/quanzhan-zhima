# 简易博客系统 · PRD

## 需求概述

构建一个极简风格的博客发布与阅读平台，专注于写作体验和阅读体验。

## 目标用户

- 技术博主（主要）
- 个人写作爱好者
- 小型团队知识库

## 核心场景

1. **文章写作**: 作者使用 Markdown 撰写文章，实时预览渲染效果
2. **文章管理**: 发布/编辑/删除文章，管理草稿和已发布内容
3. **浏览阅读**: 读者按时间线浏览文章，按标签筛选，阅读全文
4. **订阅分发**: 通过 RSS 订阅博客更新，分享文章到社交媒体

## 功能需求

### P0 (必须有)
- Markdown 文章撰写与预览
- 文章发布/编辑/删除
- 文章列表展示（时间线）
- 文章详情页阅读

### P1 (应该有)
- 标签分类管理
- RSS Feed 生成
- 代码语法高亮
- 草稿保存

### P2 (可以有)
- 评论系统
- 全文搜索
- 暗黑模式
- 社交媒体分享

## 非功能需求

- 文章页面加载 < 1 秒
- 支持 1000+ 文章存储
- 移动端响应式适配
